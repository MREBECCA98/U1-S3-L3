SIMPLE TO-DO LIST 

Questa è una semplice applicazione web che permette di creare, segnare come completate e cancellare attività. 
Ideale per esercitarsi con JavaScript, HTML e CSS.

Tecnologie utilizzate:
-HTML5 per la struttura della pagina
-CSS3 per lo styling
-JavaScript per la gestione dinamica delle attività

Funzionalità principali:
-Aggiungere nuove attività tramite un campo di input
-Segnare le attività come completate con una checkbox (testo barrato)
-Eliminare le attività tramite un pulsante “x”
-Interazione dinamica senza ricaricare la pagina

Obiettivo del progetto:
-Esercitarsi nella manipolazione del DOM con JavaScript
-Imparare a gestire eventi come click e change
-Creare una semplice applicazione interattiva e funzionale
